Luna is a Type-1 Hypervisor written in modern C++

Features:
- Hardware based virtualization with VT-x/EPT and AMD-V/NPT
- Protection against malicious devices using VT-d and AMD-Vi IOMMUs
- AHCI Disk Driver
- High Definition Audio Driver
- USB Stack + xHCI Driver
- USB Mass Storage Bulk-Only driver
- CH341 USB-to-Serial Driver
- RTL81x9 NIC Driver
- Ethernet + IPv4 + UDP TX Netstack
- Native Intel iGPU support using LIL
- VBE Modesetting using virtualization API
- Nearing ACPI compliance using LAI
